# Инструкция для пользователя

## 1) Установка на Windows
1. Скачайте установщик `ClipartGenerator-Setup-...exe`.
2. Запустите установщик и завершите установку.
3. Откройте приложение **Clipart Generator**.

## 2) Установка на macOS (без подписи Apple Developer)
1. Скачайте архив `ClipartGenerator-macOS-...zip` из Release.
2. Распакуйте архив (двойной клик по `.zip`).
3. Перетащите **Clipart Generator.app** в папку **Applications (Программы)**.
4. Первый запуск:
   - Зажмите `Control` и кликните по **Clipart Generator.app** → **Открыть**.
   - В предупреждении macOS снова нажмите **Открыть**.
5. Если macOS всё равно блокирует запуск:
   - Откройте **System Settings** → **Privacy & Security**.
   - Внизу появится сообщение о блокировке приложения.
   - Нажмите **Open Anyway (Открыть всё равно)**.

Примечание: это нужно сделать только при первом запуске после установки/обновления.

## 3) Активация лицензии
1. Откройте вкладку **Настройки**.
2. Найдите блок **Лицензия**.
3. В поле «URL сервера лицензий» введите адрес, который дал автор.
4. В поле «Лицензионный ключ» вставьте ваш ключ.
5. Нажмите **Активировать**.
6. Нажмите **Проверить**.

Если всё хорошо — лицензия активна.

## 4) Работа с приложением
1. Выберите рабочую папку.
2. Настройте режим работы.
3. Нажмите **Start**.

## 5) Обновление приложения
1. Откройте **Настройки**.
2. Нажмите **Проверить обновления**.
3. Если есть новая версия — подтвердите установку обновления.

## 6) Если не активируется ключ
Проверьте:
- правильно ли вставлен ключ,
- есть ли интернет,
- правильно ли указан URL сервера лицензий.

Если не помогло — напишите автору приложения и передайте ваш ключ.
