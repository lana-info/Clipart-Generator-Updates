# Инструкция для пользователя

## 1) Установка
1. Скачайте установщик `ClipartGenerator-Setup-...exe`.
2. Запустите установщик и завершите установку.
3. Откройте приложение **Clipart Generator**.

## 2) Активация лицензии
1. Откройте вкладку **Настройки**.
2. Найдите блок **Лицензия**.
3. В поле «URL сервера лицензий» введите адрес, который дал автор.
4. В поле «Лицензионный ключ» вставьте ваш ключ.
5. Нажмите **Активировать**.
6. Нажмите **Проверить**.

Если всё хорошо — лицензия активна.

## 3) Работа с приложением
1. Выберите рабочую папку.
2. Настройте режим работы.
3. Нажмите **Start**.

## 4) Обновление приложения
1. Откройте **Настройки**.
2. Нажмите **Проверить обновления**.
3. Если есть новая версия — подтвердите установку обновления.

## 5) Если не активируется ключ
Проверьте:
- правильно ли вставлен ключ,
- есть ли интернет,
- правильно ли указан URL сервера лицензий.

Если не помогло — напишите автору приложения и передайте ваш ключ.
