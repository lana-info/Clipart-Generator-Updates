# Чек-лист тестирования Clipart Generator

## 1) Быстрая проверка запуска
- [ ] Запустить `python main.py`
- [ ] Убедиться, что окно открывается без ошибок
- [ ] Проверить заголовок окна: есть версия (`Clipart Generator v...`)

## 2) Проверка выбора рабочей папки
- [ ] Нажать **«Выбрать папку»**
- [ ] Выбрать тестовую папку
- [ ] Убедиться, что в ней есть/создались папки `raw` и `output`

## 3) Проверка KIE API Key (замок)
- [ ] Открыть вкладку **«Настройки»**
- [ ] Убедиться, что поле `KIE API Key` по умолчанию заблокировано (`🔒`)
- [ ] Нажать на замок → должен стать `🔓`
- [ ] Проверить, что ключ можно увидеть и редактировать
- [ ] Нажать снова на замок → должен стать `🔒`
- [ ] Проверить, что поле снова заблокировано и ключ скрыт

## 4) Проверка режима «Только генерация изображений»
- [ ] Выбрать сценарий **«Только генерация изображений»**
- [ ] На вкладке **«Промпты»** добавить 2–3 промпта
- [ ] Нажать **Start**
- [ ] Проверить, что в логе появляются шаги генерации
- [ ] Дождаться завершения: сообщение содержит `Готово`
- [ ] Проверить, что появляется диалог «Открыть папку с результатами?»

## 5) Проверка кнопки Stop
- [ ] Запустить генерацию (несколько промптов)
- [ ] Дождаться, чтобы хотя бы один файл успел сохраниться (`Итог сохранён:` в логе)
- [ ] Нажать **Stop**
- [ ] Проверить, что появляется диалог «Открыть папку с уже готовыми результатами?»
- [ ] Повторить сценарий с очень ранним Stop (до `Итог сохранён:`)
- [ ] Проверить, что диалог открытия папки **не** появляется

## 6) Проверка режима «Только обработка файлов»
- [ ] Выбрать сценарий **«Только обработка файлов»**
- [ ] Нажать **«Выбрать файлы»** и выбрать 2–3 изображения
- [ ] Нажать **Start**
- [ ] Проверить лог обработки каждого файла
- [ ] После завершения проверить диалог открытия `output`

## 7) Проверка ошибок/валидации
- [ ] Очистить API key и нажать **Start**
- [ ] Убедиться, что есть понятное предупреждение про отсутствующий ключ
- [ ] Для режима обработки нажать Start без выбранных файлов
- [ ] Убедиться, что есть предупреждение про выбор файлов

## 8) Проверка сохранения настроек
- [ ] Закрыть приложение
- [ ] Снова запустить приложение
- [ ] Убедиться, что последняя рабочая папка восстановилась
- [ ] Убедиться, что настройки режима/параметров сохранились

## 9) Критерии «готово к использованию»
- [ ] Приложение стабильно запускается
- [ ] Генерация/обработка создают файлы в `output`
- [ ] Диалоги открытия папки работают корректно
- [ ] Замок ключа работает корректно (`🔒/🔓`)
- [ ] Нет падений и критических ошибок в логе
