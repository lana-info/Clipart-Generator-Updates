# Короткая инструкция для клиента (macOS)

1. Скачайте файл `ClipartGenerator-macOS-...zip` из раздела Release.
2. Распакуйте архив и перетащите `Clipart Generator.app` в папку **Applications (Программы)**.
3. Для первого запуска: зажмите `Control` → клик по приложению → **Открыть**.
4. В предупреждении macOS нажмите **Открыть** ещё раз.
5. Если macOS блокирует запуск: **System Settings → Privacy & Security → Open Anyway (Открыть всё равно)**.

Важно: это действие нужно только при первом запуске (или после обновления приложения).
